OC 面向对象概览

OC 是一种用来写 iOS 或者 OSX 程序的初级语言,它是 C 语言的超集, 继承了 C 语言所有的语法比如基本类型和流控等等. OC 在 C 语言基础上增加定义类和方法的语法, 并为对象许多运行时特性(比如动态类型, 动态绑定, 懒加载等等) 提供了语言级的支持.

​ iOS 和 OSX 程序是由一系列对象 (Object) 构成的网络组成的. 对象是类 (Class ) 的实例 .类是对象的模板或者说图纸( Blue print). 类通常由我们自定义或者由系统(Cocoa 和 Cocoa Touch )或者第三方类库提供. 我们在自定义类的时候通常会定义好公共接口(public interface) . 公共接口包含一些用来封装(encapsulate)内部数据的属性( property) 和相应的存取方法(Setter and Getter) 列表以及其他一些消息(Message)响应方法列表. 然后我们会在同一个文件或者对应的.m 文件中实现这些方法. 参考下面这段代码.

定义属性

@property 关键字会让 Xcode 在编译时候自动生成私有的变量实例(private instance variable) ,和相应的存取方法(getter setter) 

// Automatically defines a private instance variable
type _myPropertyName;
// Automatically creates a getter and setter
- (type)myPropertyName;
- (void)setMyPropertyName:(type)name;

一般命名规则自动变量名 为属性名加 -前缀, getter 方法名为属性名, setter 方法为 set 前缀加属性名. 当然也可以通过特性开关来修改变量名,我们长把返回布尔量的 getter 方法定义为下面这种形式 

@property  (getter = isEqualTOSomething) type myPeoperrty;
//生成的 getter 方法变成了下面这种形式
- (type) isEqualTOSomething;

tips:

使用 _myPropertyName 可以直接访问变量(一般用于懒加载)

使用 self.myPropertyName 通过 getter 和 setter 方法访问变量.
